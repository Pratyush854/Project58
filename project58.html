My Favourite Hobby is

Coding


ABOUT

My favourite hobby is coding. Coding is used to design games, apps and even websites. It is really fun and interesting. All apps, games and websites that we use in our quotidian life require coding. Coding is basically used to design our game, app or website and also to make the app functional. The computer doesn't understand our languages like English, French, Hindi or German, so it has its own languages like JavaScript, HTML, C++, Python and many more. Giving the computer commands to follow via its languages is called coding. I like it because with coding I can design any game which I want to play. That's all about coding.
